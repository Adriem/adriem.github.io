Why this site?

When I started developing web applications, I thought it could be great being able to host them, so people could play around with them. I started using GitHub pages, but since they don't give you access to a backend, I decided to buy a Raspberry PI 2 and host my backends there. After a bunch of hours in front of the command line interpreter, I managed to set up a web server in this little beast. Then I thought I needed a home page where I could track all my projects, so I built this page :D.
